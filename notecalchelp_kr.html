📝 노트 계산기 도움말

개요

노트처럼 자유롭게 입력하면서 실시간으로 계산할 수 있는 스마트 계산기입니다. 숫자, 수식, 변수, 블록 등을 함께 사용할 수 있습니다.

기본 수식 계산

5 * (35 - 27) / 8    → 결과: 5
25                  → 결과: 25
-7                  → 누적 결과: 23

할인 계산

$360 - 25%          → 결과: $270

변수 사용

coffee = $5
greentea = $4
sum = coffee + greentea   → 결과: $9

블록 구조

shopping_cart {
  milk: $12
  bread: $23
}

budget = $100
budget - shopping_cart    → 결과: $65

주석

// 주석입니다
; 이것도 주석입니다

팁

  • $ 기호는 선택 사항입니다.
  • 줄바꿈, 들여쓰기는 자유롭게 가능합니다.
  • 결과는 자동으로 오른쪽에 표시됩니다.
  • "Clipboard Copy" 버튼을 눌러 결과를 복사할 수 있습니다.
